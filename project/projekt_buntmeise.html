Beschreibung

Was soll realisiert werden?
Eine Quiz-App, die Fragen aus einer offenen Fragen-Datenbank anbietet.
Welche Funktion(en) soll dem Benutzer angeboten werden?
Der User kann Thema, Schwierigkeitsgrad und Anzahl an Fragen wählen. Die Fragen werden dann per API Call aus der frei verfügbaren Fragen Datenbank OpenTrivia bezogen.
Der User erhält dann die Fragen als im Multiple Choice Format (evtl. auch mit direkter Text-Eingabe). Am Ende erhält der User eine Auswertung der Ergebnisse (evtl. mit Ergebnissen der Fragen).
